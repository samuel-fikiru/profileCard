# Profile Card

A simple and visually appealing profile card component built with **HTML**, **CSS**, and **JavaScript**. This project is ideal for showcasing personal information, social media links, and other key details in a compact card layout. It's customizable, responsive, and can be easily adapted for your own portfolio, teams, or user directories.

## Features

- **Modern Design:** Clean and attractive card layout.
- **Responsive:** Works well on both desktop and mobile devices.
- **Customizable:** Easily update content, styles, and links to match your branding.
- **Interactive Elements:** Add social links and subtle hover effects using JavaScript.

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/samuel-fikiru/profileCard.git
cd profileCard
```

### Usage

1. Open `index.html` in your browser to see the profile card in action.
2. Customize the card content in `index.html` and styles in `style.css`.
3. Modify `script.js` if you want to add or change interactive features.

## File Structure

```
profileCard/
├── index.html
├── style.css
├── script.js
└── README.md
```

- **index.html** - Main HTML file for the card.
- **style.css** - CSS styles for design and layout.
- **script.js** - JavaScript for interactivity.
- **README.md** - Project documentation.

## Customization

- Update the image, name, and description in `index.html`.
- Change social links and other information as needed.
- Edit `style.css` to modify colors, fonts, and layout.
